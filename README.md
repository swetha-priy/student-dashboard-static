# 🎓 Student Dashboard

A responsive **Student Dashboard** built using **React, Vite, and Bootstrap 5**. This is a static frontend project that demonstrates a modern dashboard layout using reusable React components and Bootstrap styling.

## 📌 Features

- Responsive Header
- Navigation Bar
- Sidebar Menu
- Welcome Section
- Dashboard Statistics Cards
- Recent Activities Table
- Responsive Footer
- Clean and Modern UI
- Component-Based Architecture

## 🛠️ Tech Stack

- React.js
- Vite
- Bootstrap 5
- JavaScript (ES6)
- HTML5
- CSS3

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/student-dashboard.git
```

Go to the project folder:

```bash
cd student-dashboard
```

Install dependencies:

```bash
npm install
```

Install Bootstrap:

```bash
npm install bootstrap
```

Start the development server:

```bash
npm run dev
```
